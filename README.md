# q2
Quaternary Quantization

## Setup

1. Install [Bun](https://bun.sh/) (required).
2. Install dependencies:

```bash
bun install
```

## Scripts

- **Build** (production bundle):

```bash
bun run build
```

- **Dev** (watch mode):

```bash
bun run dev
```

- **Typecheck** (TypeScript):

```bash
bun run typecheck
```
